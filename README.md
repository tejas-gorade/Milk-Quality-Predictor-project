# ML-Based Food Adulteration Risk Predictor
An Explainable AI (XAI) approach to detecting milk adulteration using Machine Learning and IoT.

## 🚀 Overview
This project predicts the quality and safety of milk based on multiple sensor parameters (pH, Turbidity, Temperature, etc.). It uses a Random Forest classifier to categorize risk levels and SHAP (Explainable AI) to explain the reasoning behind every prediction.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib, SHAP
* **Hardware:** ESP32 Microcontroller (Planned integration)
* **Tools:** Jupyter Notebook, VS Code
